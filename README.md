data-cleaning-utils
Reusable Python utilities for cleaning and preparing datasets.

Overview  
This project provides simple, reusable functions for cleaning raw datasets prior to analysis.  
It supports tasks like removing duplicates, filling missing values, and normalizing numeric columns.

Features  
- Remove duplicate records  
- Fill missing values with a constant or column mean  
- Normalize numeric columns between 0 and 1  

Tools  
Python • Pandas • NumPy  

Example  
The included example demonstrates a small DataFrame cleaned using the utility functions.  
Users can expand this module for larger ETL or data preprocessing pipelines.
